              # 神经元模型（感知机）

              感知机 (perceptron) 是线性二分类模型，输入是实例的特征向量，输出是用 “+ 1” 和 “- 1” 表示的实例类别。作为判别模型，感知机将实例用一个超平面划分为正负两类，是神经网络和支持向量机的基础。

              • 输入：接收的信号 𝒙
              • 输出：神经元的状态 -- (兴奋 +1 / 抑制 -1)
              • 结构：

              f(x)=sign(wTx+b) f(x) = sign(w^Tx + b)\\ f(x)=sign(wTx+b)

              # 总结

              感知机对样本选择的顺序比较敏感，每次迭代样本选择的顺序不一致时，学习的感知机模型或分类超平面也往往不一样。

              感知机的收敛性：对于线性可分的数据集，感知机学习算法可以通过有限次迭代学习后达到收敛。

              # 前馈神经网络

              与 MLP 的区别

              前馈神经网络也经常被称为多层感知机（Multi Layer Perceptron, MLP），但是两者之间存在着细微差别：前馈神经网络是由多层连续的非线性函数（Sigmoid 函数）组成，而多层的感知机由多个不连续的非线性函数（符号函数）组成。

              • 各层神经元分别属于不同的层，层内无连接

              • 相邻两层之间的神经元全部两两连接

              • 整个网络中无反馈,每一层的神经元可以接收前一层神经元的信号，并产生信号输出到下一层信号从输入层向输出层单向传播。

              zl=Wlal−1+bl z^l = W^la^{l - 1} + b^l zl=Wlal−1+bl

              al=fl(zl) a^l = f_l(z^l) al=fl​(zl)

              # 训练步骤

              1. 输入数据到神经网络，得到输出值（前向传播）

              2. 通过神经网络输出和标签值得到损失值

              3. 根据损失值使用BP算法得到各个节点的梯度（反向传播）

              4. 使用优化算法更新这些节点上的参数。
